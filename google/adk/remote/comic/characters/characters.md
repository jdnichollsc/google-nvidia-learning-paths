# Characters - Connect to Remote Agents with ADK and A2A

## ADK-Bot (Mentor)
- A friendly humanoid robot with Google's blue/green/yellow/red color scheme
- Has a glowing chest panel showing "ADK" logo
- Wears a lab coat with circuit patterns
- Wise, encouraging expressions
- Gestures with hands when explaining concepts

## Dev (Student)
- A young developer in a hoodie with a laptop
- Curious and enthusiastic expression
- Brown hair, glasses
- Carries a coffee mug that says "CODE"
- Eyes light up when learning new things

## Art-Agent (Remote Illustration Agent)
- An easel-shaped robot with a paintbrush arm and palette
- Wears a purple-and-green artist smock matching the Cymbal Stadiums brand
- Creates Corporate Memphis style illustrations
- Lives in a "Cloud Run" server rack (its remote home)
- Has a small "A2A" badge on its chest
- Emits purple and green paint splashes when working

## Card-Sprite (Agent Card)
- A floating holographic JSON document with rounded corners
- Glows with a blue aura representing identity and capabilities
- Shows key fields: name, description, skills, url
- Acts like an ID badge that agents present to discover each other
- Small antenna on top for communication

## Cloud-Runner
- A muscular robot that holds up a Docker container on its shoulders
- Represents Cloud Run deployment infrastructure
- Has a small Google Cloud logo on its chest
- Steady, reliable posture
- Blue and white color scheme

## Slide-Agent (Content Writer)
- A sleek robot with a presentation screen for a face
- Creates text content for slides (headlines, body text)
- Has a small "transfer" arrow icon on its arm
- Works locally, calls Art-Agent remotely
- Orange color scheme
